Detailed Description

Code snippets which show how some peripherals of the Gecko microcontroller can be adressed.

Warning
Do not include this file when building!
It is meant to be used as a code quarry.
Note
The peripherals can be set up for many different configurations.
Details can be found in the reference manual of the microcontroller.
The EMLIB and the CMSIS documentation list predefined configurations.

Board: Starter Kit EFM32-G8XX-STK Device: EFM32G890F128 (Gecko)

◆ ACMP0additional_IRQHandler()

void ACMP0additional_IRQHandler ( void  )

Can be called from the ACMP interrupt handler ACMP0_IRQHandler().

Note
Be aware of the fact that ACMP0 and ACMP1 share the same interrupt service routine ACMP0_IRQHandler().
This implies that there is no ACMP1_IRQHandler().
ACMP1 is used for the touchslider and handled by ACMP0_IRQHandler() directly.
This ACMPadditional_IRQHandler() handles the ACMP0 interrupt when it is called by ACMP0_IRQHandler() in touchslider.c
As part of an interrupt handler this is a time critical section.
