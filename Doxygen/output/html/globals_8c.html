Convert an integer to a string (in base 10 representation)

Parameters
[in]l= integer to convert
[out]string= string representation of number l

ltostr (long to string) is the inverse function to strtol (string to long)
strtol is a standard function in stdlib.h
ltostr unfortunately is not. So it is provided here for convenience.
Make sure the string is at least [12] long to hold the maximum integer.

Note
strtol and ltostr need less memory than sscanf and sprintf
