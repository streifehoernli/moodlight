Detailed Description

Touchslider and Touchgecko.

All the functions needed to handle the touchslider and the gecko pad.
Code copied from the file "capsense.c" (version 3.20.5) from the example "C:\SiliconLabs\SimplicityStudio\v2\developer\sdks\efm32\v2\kits\common\drivers" and adapted to own requirements.

Note
The original code uses TIMER0, ACMP1, PRS and TIMER1. The code has been modified to use only TIMER1 and ACMP1.
We don't want to use TIMER1 for the touchslider as the TIMER1_CCx outputs can be used for PWM signal generation and are available on the expansion header.
We can't use TIMER2 as it might be used to boost LCD voltage. The LCD boost capacitors are connected to the TIMER2_CCx outputs.
Instead of TIMER1 it would also be possible to use the SysTick timer.
Added function CAPSENSE_getSliderValue() for freely definable range/resolution of slider position
When using ACMP0 somewhere in a project, be aware of the fact that ACMP0 and ACMP1 share the same interrupt service routine ACMP0_IRQHandler() which is placed in this file.

Board: Starter Kit EFM32-G8XX-STK Device: EFM32G890F128 (Gecko)
