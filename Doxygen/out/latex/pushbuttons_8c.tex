\doxysubsection{Detailed Description}
Pushbuttons. 

\DoxyHorRuler{0}


All the functions needed to handle the pushbuttons

\begin{DoxyNote}{Note}
The pushbutton status can be polled by reading the input. If interrupts are enabled it is also possible to use this as an event.
